# quiet
Quiet은 플레이어의 채팅을 제한할 수 있는 마인크래프트 플러그인이에요.
## 명령어
### 일반 명령어
- `/call [메시지]`: 서버에 접속 중인 모든 OP에게 메시지를 보내요.
  - `/호출 [메시지]`로도 사용할 수 있어요.
### OP 전용 명령어
- `/quiet true`: 조용한 모드를 켜요.
- `/quiet false`: 조용한 모드를 꺼요.
## 조용한 모드에 대해
조용한 모드가 켜져 있으면 OP를 제외한 모든 플레이어에게 다음 제한 사항이 적용돼요.
- 채팅을 입력할 수 없어요.
- `/call`을 제외한 어떤 명령어도 사용할 수 없어요.
